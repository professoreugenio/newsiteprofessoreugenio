Power BI – Pop-ups com Segmentações para Otimizar Espaço (Bookmarks + Seleção)

Objetivo da aula
Você vai aprender a criar um painel pop-up de filtros (segmentações de dados) no Power BI usando Indicadores (Bookmarks) e o Painel de Seleção, para liberar espaço no relatório e manter o layout limpo e profissional.

Por que usar pop-ups de filtros?

  • Evita “briga” de espaço entre gráficos, mapas e cartões.

  • Mantém filtros acessíveis sob demanda (abre/fecha).

  • Melhora usabilidade e foco visual do dashboard.

Conceitos rápidos (mini-glossário)

  • Segmentação de dados (Slicer): visual para filtrar dados por campos (Ano, Estado, Marca…).

  • Painel de Seleção: lista todos os objetos da página para mostrar/ocultar e organizar camadas.

  • Indicadores (Bookmarks): “fotos” do estado da página (o que está visível, posição, etc.), usadas para criar ações de abrir/fechar pop-ups.

  • Pop-up: container visual (geralmente uma forma com bordas arredondadas) que aparece sobre o relatório com segmentações dentro.

  • Ação (Action): configuração de um elemento (imagem/botão) para navegar para um marcador (bookmark).

Passo a passo – Construindo o pop-up de filtros

1) Preparar a área e ícones

  • Inserir → Imagem/Ícone: adicione um ícone de abrir filtros (ex.: funil/menu) e outro ícone de fechar (ex.: “X”).

  • Posicione o ícone Abrir no canto superior do relatório (fixo e visível).

2) Criar o container (forma)

  • Inserir → Formas: escolha um retângulo arredondado.

  • Ajuste raio dos cantos (nem muito reto, nem muito curvo).

  • Em Formatação do visual → Borda/Contorno: desative se quiser pop-up “clean”.

  • Preenchimento: use uma cor neutra (ex.: 20% mais escuro) para destacar do fundo.

3) Adicionar as segmentações (Ano, Estado, Marca…)

  • Visualizações → Segmentação de dados e selecione o campo (ex.: Ano).

  • Em Formatação do visual → Configurações: mude para Lista vertical (se preferir); ajuste Tamanho da fonte.

  • Copie/cole a segmentação e troque o campo para Estado e Marca.

  • Remover “(em branco)”: no painel de Filtros da segmentação → Filtro avançado → “não está em branco” → Aplicar.

4) Harmonizar visual (cores e textos)

  • Tela de fundo da segmentação = mesma paleta do container; Fonte clara (branco) para contraste.

  • Cabeçalho com cor de destaque (ex.: amarelo) e aplique com o Pincel de formatação nas demais segmentações.

5) Alinhar, distribuir e agrupar

  • Selecione as segmentações (Shift + clique) → Formato → Alinhar/Distribuir (distâncias iguais).

  • Selecione container + segmentações (sem o ícone Abrir) → Formato → Agrupar.

  • Coloque o ícone Fechar dentro do pop-up, no canto superior.

6) Painéis necessários

  • Exibir → Painel de Seleção e Painel de Indicadores.

  • No Painel de Seleção, renomeie o grupo para “Painel Pop-up”.

7) Criar os Indicadores (Bookmarks)

  • Estado “Fechado” (OFF):

    • No Painel de Seleção, oculte o grupo Painel Pop-up (olhinho).

    • No Painel de Indicadores, Adicionar → renomeie para “Painel OFF”.

    • Desmarque “Dados” no indicador (para não limpar filtros ao alternar).

  • Estado “Aberto” (ON):

    • Exiba o grupo Painel Pop-up.

    • Adicionar novo indicador → “Painel ON”.

    • Desmarque “Dados” aqui também.

8) Ações nos ícones

  • Ícone Abrir → Formatação do visual → Ação → Tipo: Marcador → Marcador: “Painel ON”.

  • Ícone Fechar → Ação → Marcador: “Painel OFF”.

9) Teste

  • Clique no ícone Abrir (pop-up aparece).

  • Clique no ícone Fechar (pop-up some).

  • Aplique um filtro e feche o pop-up: os filtros devem permanecer (porque “Dados” foi desmarcado nos indicadores).

Boas práticas

  • Crie padrão visual (cores e fontes) e reutilize com o Pincel.

  • Use poucos níveis de filtros por pop-up para não “pesar” a experiência.

  • Nomeie objetos e indicadores de forma clara: “Popup_Filtros_ON/ OFF”.

Checklist rápido

  • Container criado e formatado

  • Segmentações configuradas (sem “em branco”)

  • Itens alinhados, distribuídos e agrupados

  • Painéis de Seleção e Indicadores abertos

  • Indicadores ON/OFF criados sem “Dados” marcado

  • Ações nos ícones funcionando

Exercício prático

  1. Monte um pop-up com 3 segmentações (Ano, Estado, Marca).

  2. Crie ícones de abrir/fechar e configure Indicadores ON/OFF.

  3. Aplique filtros, feche e reabra para validar que filtros permanecem.

  4. Padronize cores com o Pincel.
